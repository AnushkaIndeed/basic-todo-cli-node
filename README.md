# 📝 Todo CLI App (Node.js)

A simple **command-line todo list app** built with **Node.js**.  
You can add tasks, view tasks, and store them in a `tasks.json` file.  

---

## 🚀 Features
- Add new tasks
- List all tasks
- Saves tasks in `tasks.json`
- Runs directly from the terminal
